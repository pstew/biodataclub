## Shared Resources Travel Agent

### Project Lead: [Richie Reich](mailto:richie.reich@moffitt.org)

**Background**: 
Investigators at Moffitt don't always know where to go or who to contact for
help from one of the many available shared resources.  This tool will use
generative AI to discern needs from conversational prompts with the purpose of
connecting investigators to the right shared resource(s).
 
**Project Significance**: 
This project can help Moffitt researchers identify the best way to get help from
Shared Resources. It will help facilitate access to and increase project work

## A Directory of Immune and Oncological Studies (ADIOS) Cancer

### Project Lead: [Joshua Davis](mailto:JOSHUA.DAVIS@moffitt.org)

**Background**: 
We have established an initial repository of publicly available scRNAseq,
RNAseq, and microarray data https://biostools.moffitt.org/4472414/Shiny/ADIOS_Cancer/. 
We have established several resources for real-time analysis of these public
data sets. We aim to expand the resource with 50 studies over the course of the
hackathon.
 
**Project Significance**: 
The repository will facilitate easy access to public data for the Moffitt
community. This will also facilitate the generation of a gene signature specific
to IO treatment, drug response, and cell types for downstream bioinformatics
analysis.
 
**Technical Specifications or Programming background**: 
Learn the basics of bioinformatics gene expression analysis.
R script, R Shiny app, and Web Development

## Utility of generative AI for generation of patient informational videos

### Project Leads: [Jamie Teer](mailto:jamie.teer@moffitt.org) and [Christine Vinci](mailto:christine.vinci@moffitt.org)

**Background**: 
Engagement, education, and interaction with the public is an important part of the Moffitt mission. 
Communicating with both cancer patients and the community as a whole is critical to share important information regarding the prevention and cure of cancer.
We hypothesize that the use of generative AI may be a useful approach in the process of developing communication videos,
and this project aims to explore the utility of generative technologies in creating actual videos, storyboard templates, 
or ideas relating to communication with the community.
 
**Project Significance**: 
1. Utilize open-source generative AI models to explore video generation:
      - Text to video
      - Text to image
      - Image to video
2. Develop knowledge regarding effective prompts to generate video
      - explore value of using Large Language Models to assist with prompt generation and tuning
3. Determine practical use cases for video generation using AI

 
**Technical Specifications or Programming background**: 
1. Utilize the open source Wan2.2 5B model
2. Project lead will initiate ComfyUI server using the red HPC cluster
3. Project members will connect to server via web browser
4. Project members will utilize ComfyUI graphical interface to enter prompts, generate video, and review videos
